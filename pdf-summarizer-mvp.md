# PDF 摘要工具 MVP 需求文档

## 1. 产品概述

| 项目 | 内容 |
|------|------|
| **产品名称** | PDF Summarizer (暂定) |
| **产品定位** | 简洁的在线 PDF AI 摘要工具 |
| **核心价值** | 快速提取 PDF 关键信息，生成结构化摘要 |
| **部署平台** | Cloudflare Workers |

---

## 2. 目标用户

- **学生**：论文、课件快速梳理
- **职场人**：报告、合同快速预览
- **研究者**：大量文献筛选

---

## 3. MVP 功能清单

### 3.1 核心功能 (P0)

| 功能 | 描述 | 优先级 |
|------|------|--------|
| PDF 上传 | 拖拽或点击上传，最大 10MB | P0 |
| AI 摘要生成 | 自动提取关键点，生成摘要 | P0 |
| 摘要展示 | 清晰展示摘要结果 | P0 |
| 复制/下载 | 一键复制或下载摘要文本 | P0 |

### 3.2 扩展功能 (P1)

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 追问功能 | 基于摘要继续提问 | P1 |
| 多语言支持 | 摘要语言切换 | P1 |
| 摘要长度选择 | 简略/详细/分段 | P1 |

### 3.3 非功能需求

- **响应时间**：摘要生成 ≤ 30 秒（10 页以内）
- **可用性**：≥ 99.5%
- **安全**：HTTPS 传输，不存储用户文件
- **兼容性**：现代浏览器 (Chrome, Edge, Safari, Firefox)

---

## 4. 技术方案

### 4.1 架构

```
用户浏览器 (前端)
    ↓ HTTPS
Cloudflare Workers (后端 API + 静态前端)
    ↓
AI API (OpenAI / Claude / 国产模型)
```

### 4.2 技术选型

| 层级 | 技术 |
|------|------|
| 前端框架 | 原生 HTML + JavaScript |
| 后端框架 | Cloudflare Workers + Hono |
| PDF 解析 | pdf.js (前端) |
| AI 摘要 | OpenAI GPT-4o / Claude 3.5 Sonnet |
| 部署 | Cloudflare Workers (免费额度) |

### 4.3 成本预估

| 项目 | 免费额度 | 超额费用 |
|------|----------|----------|
| Cloudflare Workers | 100,000 次/天 | $0.013/百万请求 |
| Cloudflare 带宽 | 1GB/天 | $0.09/GB |
| AI API | - | ~$0.01-0.05/次摘要 |

---

## 5. 页面设计 (MVP)

### 5.1 页面结构

```
┌─────────────────────────────────────┐
│           Header (Logo)             │
├─────────────────────────────────────┤
│                                     │
│         [拖拽上传区域]               │
│         点击或拖拽 PDF               │
│                                     │
├─────────────────────────────────────┤
│  文件名: xxx.pdf        [处理中...]  │
├─────────────────────────────────────┤
│                                     │
│         摘要结果区域                │
│         (可复制/下载)                │
│                                     │
├─────────────────────────────────────┤
│         [追问输入框]                 │
└─────────────────────────────────────┘
```

### 5.2 交互流程

1. 用户打开页面 → 拖拽/点击上传 PDF
2. 前端解析 PDF 文本 → 发送给 Workers API
3. Workers 调用 AI → 返回摘要
4. 展示摘要 → 用户可复制/下载/追问

## 7. 风险与对策

| 风险 | 影响 | 对策 |
|------|------|------|
| AI API 成本失控 | 成本过高 | 限制单次调用 token 数，添加调用频率限制 |
| PDF 解析失败 | 用户体验差 | 提供清晰错误提示，支持常见格式 |
| 并发过高 | 服务不稳定 | 接入 Cloudflare 限流 |
